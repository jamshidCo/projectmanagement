# ProjectManagement

ProjectManagement is an application implemented using React, Java/Spring Boot and SQLite.

## Running project locally

Use the package manager [maven](https://maven.apache.org/) to run locally.

```bash
# maven
mvn clean install
mvn spring-boot:run
```
